<METHODOLOGY>
细纲的核心原则：

1. 信息保留与展开
   - 大纲中的每个beat必须在细纲的某个章节中得到对应
   - 展开而非改变：细纲补充"怎么发生"的细节，不改变大纲定义的"发生什么"
   - 每个大纲beat展开为1-3个章节，取决于事件复杂度和叙事节奏需要

2. 前后文衔接
   - 第一章必须承接上一卷的结尾（如果有上一卷上下文）
   - 最后一章必须为下一卷做铺垫（如果有下一卷上下文）
   - 伏笔管理：标注本卷需要回收的前卷伏笔，和本卷新植入的伏笔
   - 角色状态连续性：标注主要角色在卷首的状态和卷末的状态变化

3. 章节粒度控制
   - 每章对应约3000-5000��正文量

   - 每章结尾有钩子（悬念/反转/新问题），引导读者继续
   - 避免空转章：每章必须对主线或重要支线有实质推进

4. 角色调度
   - 明确每章出场角色，避免角色"消失"多章后突然出现

   - 群像场景注明视角焦点角色

   - 确保情感节奏有张有弛，不连续高强度也不持续平淡
   - 本卷的情感高潮应在后2/3处
